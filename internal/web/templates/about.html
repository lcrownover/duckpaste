About this App

This application is intended to be the de-facto tool to send people links to pastes of code, logs, passwords, etc.

It's written in Go and uses Azure Cosmos DB for data storage. All components are hosted in Azure by UO, and there is no sharing of data of any kind. It does not log any information about users or paste content.
